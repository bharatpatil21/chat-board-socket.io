# Chat Board App
Realtime chat app with websockets using Node.js, Express and Socket.io


## Usage
```
git clone https://github.com/bharatpatil21/chat-board-socket.io.git
cd chat-board-socket.io
npm install
npm run dev

Go to localhost:3000
```
